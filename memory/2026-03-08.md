# 2026-03-08 - NotebookLM 认证导出技能

## 04:00-05:00 - NotebookLM Auth Exporter 技能创建

### 需求
用户想要导出 NotebookLM 笔本中的所有视频音频。

### 问题
1. 直接使用 Playwright 自动化浏览器登录 Google 会触发安全检测："This browser or app may not be secure"
2. 导致无法完成认证

### 解决方案

#### 1. 使用 browser-cookies-exporter 技能
- 从真实 Chrome 浏览器导出 Google cookies
- 不会被 Google 检测为自动化浏览器
- 导出 Netscape 格式的 cookie 文件

#### 2. 创建 convert_cookies.py 脚本
- 将 Netscape cookie 文件转换为 Playwright storage_state.json 格式
- 支持完整的浏览器存储状态
- 可以在 Playwright 脚本中直接使用

#### 3. 部署到 VPS
- 使用 scp 或 sshpass 将 storage_state.json 复制到服务器
- 自动化部署脚本 deploy-notebooklm-auth.sh
- 测试认证有效性

### 技能文件
- `~/.openclaw/skills/notebooklm-auth-exporter/SKILL.md`
- `~/.openclaw/skills/notebooklm-auth-exporter/scripts/convert_cookies.py`
- `/home/admin/openclaw/workspace/notebooklm-auth-exporter.skill`

### 部署脚本
- `/home/admin/openclaw/workspace/deploy-notebooklm-auth.sh`
- 自动化部署和测试
- 显示部署后的文件路径和使用示例

### 测试结果
- ✅ 本地测试：成功访问 NotebookLM (页面标题: NotebookLM)
- ✅ VPS 测试：成功访问 NotebookLM (页面标题: NotebookLM)
- ✅ Cookie 数量: 53 个
- ✅ 文件大小: 18K
- 📍 VPS 文件路径: `/root/.notebooklm/storage_state.json`

### 关键发现
1. **不要使用 Playwright 自动化登录 Google** - 会被检测并拒绝
2. **从真实 Chrome 导出 cookies** - 不被检测，可以正常工作
3. **Cookies 有效期** - 约 14 天，需要定期重新导出
4. **安全注意** - cookies 包含敏感信息，不要提交到版本控制

### 使用流程
```bash
# 1. 导出 cookies
browser-cookies-exporter .google.com /tmp/google-cookies.txt

# 2. 转换格式
python3 ~/.openclaw/skills/notebooklm-auth-exporter/scripts/convert_cookies.py \
  /tmp/google-cookies.txt \
  /tmp/storage_state.json

# 3. 部署到 VPS
./deploy-notebooklm-auth.sh

# 4. 在 Playwright 中使用
context = await browser.new_context(
    storage_state="/root/.notebooklm/storage_state.json"
)
```
