### 3. VPS SSH 密钥复制到本地
- **目的：** 实现本地直接推送到 GitHub，无需 VPS 中转
- **密钥类型：** ed25519
- **本地密钥：** ~/.ssh/id_ed25519_github

#### 配置步骤
1. 从 VPS 下载 SSH 密钥（私钥和公钥）
2. 设置密钥权限（600/644）
3. 测试 GitHub 连接
4. 配置 Git 使用本地 SSH 密钥
5. 切换远程仓库从 HTTPS 到 SSH

#### 测试结果
- ✅ GitHub 连接成功
- ✅ Git 配置完成
- ✅ 直接推送成功
