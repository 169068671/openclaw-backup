- 视频编码：AV01 (H.266)

- 文件位置：/tmp/Cooking 3： Cara membuat bubur mutiara dengan metode 5.30.7 (menit).mp4
- 文件大小：68M

**下载命令**：
```bash
yt-dlp --proxy socks5://127.0.0.1:1080 \
  --cookies-from-browser chrome \
  --js-runtimes node \
  --remote-components ejs:github \
  -f "bestvideo[ext=mp4]+bestaudio[ext=m4a]/best[ext=mp4]/best" \
  -o "/tmp/%(title)s.%(ext)s" \
  "https://www.youtube.com/watch?v=a5SXtaF7Kt0"
```

**关键技术点**：
- 必须使用 SOCKS5 代理（SSH 隧道）
- 必须使用浏览器 cookies
- 必须使用 Node.js JavaScript runtime
- 必须使用 GitHub EJS 远程组件解决 YouTube 挑战

**耗时**：约 32 秒（视频 30s，音频 2s）

### 09:38 - 导出 YouTube 浏览器 Cookies（Netscape 格式）

**任务**：导出 Chrome 浏览器的 YouTube cookies，保存为 Netscape 格式到桌面

**完成情况**：
- ✅ 安装 browser-cookie3（Python 库）
- ✅ 导出 21 个 cookies
- ✅ 保存为 Netscape 格式
- ✅ 文件位置：/home/admin/Desktop/cookies-youtube.txt
- ✅ 文件大小：3.1K

**导出命令**：
```bash
# 安装 browser-cookie3
pip3 install --user browser-cookie3

# 导出 cookies
python3 -c "
import browser_cookie3
import http.cookiejar as cj

cookies = browser_cookie3.chrome(domain_name='.youtube.com')
cookie_jar = cj.MozillaCookieJar('/home/admin/Desktop/cookies-youtube.txt')
for cookie in cookies:
    cookie_jar.set_cookie(cookie)
cookie_jar.save()
"
```

**使用方法**：
```bash
# 使用导出的 cookies 下载 YouTube 视频
yt-dlp --proxy socks5://127.0.0.1:1080 \
  --cookies "/home/admin/Desktop/cookies-youtube.txt" \
  --js-runtimes node \
  --remote-components ejs:github \
  -f "bestvideo[ext=mp4]+bestaudio[ext=m4a]/best[ext=mp4]/best" \
  -o "/tmp/%(title)s.%(ext)s" \
  "https://www.youtube.com/watch?v=XXX"
```
