# 📖 第二部分：青铜时代 - Python与3D的初次碰撞

## 2.1 你好，控制台！第一行创世代码

不用害怕编程！在FreeCAD的Python控制台中，一行代码就能凭空变出一个立方体。

```python
# 导入FreeCAD的零件模块
import Part
# 创造一个长宽高都是10的立方体
my_box = Part.makeBox(10, 10, 10)
# 把立方体扔进当前文档里显示出来
Part.show(my_box)
```

🎉 **魔法生效**：屏幕上出现了一个正方体！这就是你的"Hello World"！

## 2.2 循环与条件：打造阶梯与彩色迷宫

利用Python的 `for` 循环，我们可以瞬间造出100个台阶，再也不用手动复制粘贴到手酸。

```python
import FreeCAD as App
import Part

doc = App.ActiveDocument

# 用for循环造5个台阶
for i in range(5):
    box = Part.makeBox(10, 10, 10)
    obj = doc.addObject("Part::Feature", f"Step_{i}")
    obj.Shape = box
    # 计算每个台阶的位置 (x, y, z)
    obj.Placement.Base = App.Vector(i*10, i*10, i*10)

doc.recompute() # 刷新一下，见证奇迹！
```

# 📖 第三部分：白银时代 - 打造专属控制台 (FreeCAD GUI)

高中生最喜欢什么？当然是拥有一个属于自己的软件控制面板！我们将使用 **PySide (Qt GUI)** 在FreeCAD里画一个窗口。代码不难，跟着做就行！

## 3.1 告别黑框：开启图形界面 (GUI) 之门

让我们写一个带按钮的小窗口。点击按钮，它就会在三维世界里生成一个球体。

```python
from PySide import QtGui
import FreeCAD as App
import Part

class MyMagicWindow(QtGui.QWidget):
    def __init__(self):
        super(MyMagicWindow, self).__init__()
        self.initUI()

    def initUI(self):
        # 设置窗口标题和大小
        self.setWindowTitle('我的魔法造物台')
        self.setGeometry(100, 100, 200, 100)

        # 添加一个按钮
        self.btn = QtGui.QPushButton('生成魔法球!', self)
        self.btn.clicked.connect(self.make_sphere) # 绑定点击事件

        # 设置布局
        layout = QtGui.QVBoxLayout()
        layout.addWidget(self.btn)
        self.setLayout(layout)

    def make_sphere(self):
        # 按钮按下的魔法逻辑
        sphere = Part.makeSphere(15)
        Part.show(sphere)
        App.Console.PrintMessage("魔法球已生成！\n")

# 显示窗口
my_gui = MyMagicWindow()
my_gui.show()
```

## 4.1 碰撞检测：物体间的"亲密接触"

做个简单的物理模拟吧！当一个移动的方块碰到另一个圆柱体时，圆柱体会变成红色报警！
我们通过检测两个物体 `Shape.BoundBox`（边界框）是否相交来实现：

```python
# 核心碰撞检测逻辑（伪代码演示）
box_bbox = obj_box.Shape.BoundBox
cylinder_bbox = obj_cylinder.Shape.BoundBox

if box_bbox.intersect(cylinder_bbox):
    # 如果撞上了，把圆柱变成红色
    obj_cylinder.ViewObject.ShapeColor = (1.0, 0.0, 0.0) # RGB中的红色
else:
    # 安全距离，保持绿色
    obj_cylinder.ViewObject.ShapeColor = (0.0, 1.0, 0.0)
```

# 📖 附录：打怪升级避坑指南 (开发者日记)

作为高中生，在实现过程中一定会遇到Bug。这份文档记录了我们在教学和开发过程中收集的"血泪史"：

### 6.1 常见 Bug 与修复秘籍

1. **报错 `IndentationError` (缩进错误)**：
   * *症状*：代码一运行就红字报错。
   * *解药*：Python是靠缩进认"一家人"的。检查你的 `for` 循环和 `if` 语句下面是不是严格空了4个空格。**不要混用Tab和空格！**

2. **FreeCAD卡死/未响应**：
   * *症状*：运行了一个死循环 `while True:` 且没有设置跳出条件，或者生成的阵列数量太大（如 `range(10000)`）。
   * *解药*：在写动画或密集计算时，记得加上 `App.updateGui()` 或者使用 QT定时器（QTimer），不要堵塞主线程！

3. **GUI 窗口闪退**：
   * *症状*：写了UI代码，运行后窗口闪了一下就不见了。
   * *解药*：变量被垃圾回收了。确保你的窗口实例是一个全局变量（如上面代码中的 `my_gui` 而不是放在函数内部的局部变量）。

### 6.2 研发过程记录

* *第一阶段（需求调研）*：发现大部分学生觉得背Python语法枯燥，决定引入FreeCAD，用可视化的3D反馈刺激学习多巴胺。
* *第二阶段（工具选型）*：放弃了Tkinter，全面转向PySide（Qt），因为FreeCAD原生集成，兼容性最好。
* *第三阶段（课程迭代）*：在第11周教学中发现，参数化设计必须结合界面，于是增加了"定制GUI面板"环节，学生好评率上升至95%！

## 文件 6：`chap2/2.2-loop.md` (循环与条件控制)

```markdown
# 🔄 2.2 循环的魔法：自动阵列与彩色台阶

如果你要画 100 级台阶，手动拉伸要多久？10 分钟？用 Python，只需要 0.1 秒。

### 📐 数理原理：等差数列与空间平移
我们要创造一段台阶，每一级台阶相对于上一级，在 X 轴和 Z 轴（高度）上都要增加固定的距离。这在数学上就是一个简单的等差数列映射到空间向量上。

### 💻 FreeCAD 脚本：循环生成台阶
```python
import FreeCAD as App
import Part
from FreeCAD import Vector

doc = App.ActiveDocument if App.ActiveDocument else App.newDocument()

# 制作指南：用 for 循环造 10 级台阶
for i in range(10):
    # 造一个踏板 (长20, 宽10, 高2)
    step = Part.makeBox(20, 10, 2)
    obj = doc.addObject("Part::Feature", f"Step_{i}")
    obj.Shape = step

    # 物理原理：台阶依次向上(Z)和向前(Y)平移
    # x不变, y每次增加10, z每次增加2
    obj.Placement.Base = Vector(0, i*10, i*2)

doc.recompute()
```

### 📋 实验报告

* **调试与反思**：在运行过程中，`i` 的值从 0 变到了 9。计算机忠实地执行了 10 次平移操作。试想一下，如果在代码里加入 `if` 条件语句，比如"遇到偶数台阶就在上面挖个洞"，是不是就能瞬间生成复杂的镂空艺术品了？

# 🎛️ 3.1 白银时代：开启图形界面 (GUI) 之门

真正的极客从不满足于黑框框！在这一节，王老师带大家利用 Qt (PySide) 为你的模型编写一个可视化的"遥控器"。

### 📐 运行原理：事件驱动机制
图形界面（GUI）的核心原理是"事件驱动"。我们画一个按钮，它平时静静地待着；当我们"点击（Click）"它时，就会触发一个"事件"，从而调用我们写好的 3D 生成脚本。

### 💻 FreeCAD 脚本：我的专属造物面板
将这段代码在 Python 控制台中粘贴运行，不要漏掉任何缩进哦！

```python
from PySide import QtCore, QtGui
import FreeCAD as App
import Part

# 制作指南：定义我们自己的控制面板类
class CreatorWindow(QtGui.QWidget):
    def __init__(self):
        super(CreatorWindow, self).__init__()
        self.initUI()

    def initUI(self):
        self.setWindowTitle('东台中学-魔法造物台')
        self.setGeometry(300, 300, 250, 100)

        # 1. 创造一个按钮
        self.btn = QtGui.QPushButton('✨ 生成魔法球', self)
        # 2. 绑定事件：点击按钮 -> 召唤球体
        self.btn.clicked.connect(self.spawn_sphere)

        # 3. 设置垂直布局
        layout = QtGui.QVBoxLayout()
        layout.addWidget(self.btn)
        self.setLayout(layout)

    def spawn_sphere(self):
        # 点击后执行的动作
        sphere = Part.makeSphere(15)
        Part.show(sphere)

# 启动面板

my_panel.show()

```markdown
# 🎚️ 3.2 掌控维度：用滑动条实时控制模型

接下来，我们要玩点更高级的——实时物理互动。我们将制作一个滑动条（Slider），指尖滑动，模型的大小或旋转角度跟着实时变化。

### 📐 物理与数学原理：线性映射
滑动条有一个最小值（如0）和最大值（如100）。我们需要通过数学公式，将这 0~100 的整数，**线性映射**到物体的旋转角度（0°~360°）或大小尺寸上。

### 💻 FreeCAD 脚本：动态缩放立方体
```python
from PySide import QtCore, QtGui
import FreeCAD as App
import Part
from FreeCAD import Vector

class SliderWindow(QtGui.QWidget):
    def __init__(self):
        super(SliderWindow, self).__init__()
        # 先在场景里准备一个待控制的方块
        self.doc = App.ActiveDocument if App.ActiveDocument else App.newDocument()
        self.box_obj = self.doc.addObject("Part::Feature", "DynamicBox")
        self.box_obj.Shape = Part.makeBox(10, 10, 10)
        self.initUI()

    def initUI(self):
        self.setWindowTitle('维度控制器')
        self.setGeometry(300, 300, 300, 150)

        # 创建滑动条 (水平方向)
        self.slider = QtGui.QSlider(QtCore.Qt.Horizontal, self)
        self.slider.setMinimum(10) # 最小尺寸 10
        self.slider.setMaximum(100) # 最大尺寸 100

        # 绑定滑动事件 (valueChanged)
        self.slider.valueChanged.connect(self.on_slide)

        layout = QtGui.QVBoxLayout()
        layout.addWidget(QtGui.QLabel("调节方块大小:"))
        layout.addWidget(self.slider)
        self.setLayout(layout)

    def on_slide(self, val):
        # 当滑动条数值改变时，重新生成不同尺寸的方块
        self.box_obj.Shape = Part.makeBox(val, val, val)
        self.doc.recompute()

# 呼出控制器
slider_gui = SliderWindow()
slider_gui.show()
```

### 📋 实验报告与制作指南

* **互动体验**：拖动滑动条，感受三维空间中的方块如同具备生命一般呼吸、膨胀。这种视觉反馈对于我们理解**"参数即形态"**的工程理念至关重要！
* **拓展挑战**：试着将 `Part.makeBox` 替换为修改物体的 `Placement.Rotation`（旋转），你就能制作出一个可以控制机械臂旋转的监控台。
```

# 🖨️ 5.1 创意落地：数字到实体的跨越

代码的世界很精彩，但能够拿在手中摩挲的实物更令人激动。我们将运用物理常识，完成模型的切片与打印。

### 📐 物理原理：热熔融沉积 (FDM) 与重力约束
常见的 3D 打印机基于 FDM 原理：将塑料耗材加热融化，一层一层地挤出并冷却固化。
**注意重力陷阱**：在物理世界中，材料不能悬空打印。如果你的代码生成了大于 45° 的悬垂结构（比如悬空的屋顶），我们必须在切片时加入"支撑（Support）"结构。

### 💻 制作指南：STL 导出与切片
1. **导出网格**：
   在 FreeCAD 中选中你的模型，点击菜单栏 `文件 -> 导出`，格式选择 **STL Mesh (*.stl)**。
2. **切片软件 (Cura / 3D One 打印模块)**：
   将 STL 导入切片软件。设置三个关键参数：
   * **层高 (Layer Height)**：0.2mm（精度与速度的最佳平衡）。
   * **填充密度 (Infill)**：15% ~ 20%（兼顾物理强度和节省耗材）。
   * **支撑 (Support)**：如果模型有大角度悬空，务必勾选"生成支撑"。
