## ⚠️ 已知限制

1. **MiniMax API 集成未完成**
   - 当前只生成模板图片
   - 需要实现实际 API 调用
   - 需要配置模型参数（分辨率、风格等）

2. **参考图片功能未实现**
   - `--image` 参数已定义
   - 但实际处理逻辑未实现
   - 提示词中的 `@ref.jpg` 引用未解析

3. **编辑功能未实现**

4. **错误处理需要改进**
