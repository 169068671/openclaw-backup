## 📋 项目概述

基于 MiniMax API 的轻量级分镜生成工具，支持 9 宫格/25 宫格分镜生成、图片切割、编辑与合并导出。

### 核心目标

✅ **轻量级设计** - 无需桌面应用，命令行操作
✅ **分镜生成** - 支持 3x3 (9 宫格) 和 5x5 (25 宫格)
✅ **图片处理** - 切割、编辑、合并一体化
✅ **参考图片** - 支持上传参考图片保持风格一致
✅ **多风格支持** - 真实、动漫、3D、卡通风格

## 🚀 使用指南

### 安装

```bash
# 1. 安装依赖
pip3 install --user requests Pillow numpy

# 2. 配置 API Key
storyboard-generator config <API Key>

# 3. 开始使用
storyboard-generator generate --grid 3x3 --prompts "镜头1|镜头2|...|镜头9"
```

### 快速开始

```bash
# 生成分镜
storyboard-generator generate \
  --grid 3x3 \
  --prompts "镜头1：城市天际线|镜头2：街道视角|镜头3：人物特写|..." \
  --output my_storyboard.jpg

# 切割分镜
storyboard-generator cut my_storyboard.jpg

# 合并分镜
storyboard-generator merge ./cuts --grid 3x3 --output final.jpg
```

### 查看帮助

```bash
storyboard-generator --help
storyboard-generator generate --help
storyboard-generator cut --help
storyboard-generator merge --help
```
