## 使用建议

### 国内音乐

**推荐平台**：
- ✅ 网易云音乐：支持无损，歌单丰富
- ✅ QQ音乐：歌单丰富，品质好
- ✅ 酷狗音乐：搜索准确
- ✅ 酷我音乐：无损支持

**示例**：
```bash
# 网易云音乐
musicdl 歌曲 -p netease -q lossless

# QQ音乐
musicdl 歌曲 -p qq -q 320
```

### 国际音乐

**推荐平台**：
- ✅ YouTube：视频转音频
- ✅ Spotify：全球最大音乐平台
- ✅ Apple Music：苹果生态

**示例**：
```bash
# YouTube
musicdl song -p youtube

# Spotify
musicdl song -p spotify
```

### 有声读物

**推荐平台**：
- ✅ 喜马拉雅：最大有声平台
- ✅ 懒人听书：小说丰富
- ✅ 蜻蜓FM：播客聚合

**示例**：
```bash
# 喜马拉雅
musicdl 有声书 -p ximalaya

# 懒人听书
musicdl 小说 -p lazy
```
