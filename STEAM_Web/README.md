# 🚀 欢迎来到"代码造物"数字工坊！

> **"在比特与原子的交界处，用代码定义万物。"**

欢迎来到江苏省东台中学通用技术校本课程专属平台！本项目源自《STEAM 理念下基于编程建模的高中 3D 打印校本课程的开发与研究》（盐城市教育学会课题编号：2024B-178）。

在这里，我们摒弃传统的"拉拽式"3D建模，将 **STEAM**（科学、技术、工程、艺术、数学）理念与 **FreeCAD Python 编程**深度融合。你将化身为数字造物主，不仅能用代码生成复杂的几何形态，还将学习使用 **Qt GUI** 开发出带有按钮、滑动条的专属交互面板，最终将这些数字创意通过 3D 打印机转化为改变生活的真实物理产品！

---

## 🌟 在这里，你将经历：

### 🏛️ 青铜时代：Python 与三维空间的碰撞
写下第一行代码，召唤出三维世界的纯粹几何体。理解变量、循环、条件判断如何操控三维空间。

### ⚪ 白银时代：打造专属控制台
告别枯燥的黑框，用 Python 设计出带有按钮、输入框和滑动条的**专属图形交互界面 (GUI)**。体验事件驱动编程的魅力。

### 🌟 黄金时代：让世界动起来
将代码注入灵魂，利用 QTimer 定时器和三角函数，让几何体根据物理原理和数学公式动起来。

### 🏆 王者时代：创意落地与实物化
切片、上机、打印！见证数字代码变成手中实物的奇迹。通过项目答辩和开源分享，成为一名真正的极客创客。

## 🛠️ 快速开始

### 第一步：准备工具

1. **下载 FreeCAD**：访问 [FreeCAD 官网](https://www.freecad.org/) 下载并安装
2. **调出控制台**：启动 FreeCAD 后，点击 `视图` → `面板` → 勾选 `Python 控制台`
3. **选择工作台**：在顶部工具栏选择 `Part` 或 `PartDesign` 工作台

### 第二步：开始学习

点击左侧导航栏，从"第1课：造物新纪元"开始你的造物之旅！每节课都包含：
- 📐 **物理/数理原理**：连接高中物理、数学知识
- 💻 **FreeCAD 脚本**：可立即运行的代码示例
- 📋 **实验报告**：引导观察、记录、思考
- 🛠️ **制作指南**：分步骤操作说明与拓展挑战

## 💡 为什么选择"代码造物"？

### 传统建模 vs 编程建模

| 对比项 | 传统"拉拽式"建模 | Python 编程建模 |
|--------|----------------|----------------|
| 参数修改 | 手动调整多个参数，耗时 | 修改变量，模型自动更新 ⚡ |
| 复杂结构 | 手动绘制，易出错 | 循环生成，瞬间完成 🚀 |
| 重复任务 | 复制粘贴，繁琐 | 函数封装，一键调用 ✨ |
| 学习价值 | 软件操作技能 | 编程思维 + 空间想象力 + 数理逻辑 🧠 |

### STEAM 融合体验

- **科学 (Science)**：力学原理、简谐振动、碰撞检测
- **技术 (Technology)**：Python 编程、FreeCAD 建模、Qt GUI 开发
- **工程 (Engineering)**：参数化设计、结构优化、系统集成
- **艺术 (Art)**：造型美学、色彩设计、生成艺术
- **数学 (Mathematics)**：函数、坐标、几何、算法、向量运算
