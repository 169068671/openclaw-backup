## 📐 物理/数理原理

### 线性映射

在机械工程中，连杆和铰链的运动是**连续的**。

在计算机中模拟这种连续物理状态时，我们需要建立：
- **UI 控件数值**（如滑动条的 0~100）
- **物体物理属性**（如长度、旋转角度）

之间的**线性映射关系**：

```
UI 数值 ──────── 公式 ──────── 物理属性
  0-100      映射转换      10-150mm
```

### 映射公式

假设：
- 滑动条范围：`min_val` ~ `max_val`（如 10~100）
- 物理属性范围：`min_phys` ~ `max_phys`（如 10mm~150mm）

映射公式：
```python
# UI 值转物理值
physical_val = min_phys + (ui_val - min_val) * (max_phys - min_phys) / (max_val - min_val)
```

### 事件驱动机制

GUI（图形用户界面）的核心是**事件驱动**：
1. UI 处于**监听状态**
2. 用户操作（如拖动滑动条）→ 触发**事件**
3. 事件 → 调用**回调函数**
4. 回调函数 → 执行具体的 3D 操作

这就像是：
- 你按下遥控器按钮
- 遥控器发送信号
- 电视机收到信号并换台

## 💻 FreeCAD 脚本

让我们创建一个可以用滑动条实时控制长方体的界面：

```python
from PySide import QtCore, QtGui
import FreeCAD as App
import Part

class SliderWindow(QtGui.QWidget):
    """维度控制器：用滑动条控制长方体的大小"""

    def __init__(self):
        super(SliderWindow, self).__init__()

        # 创建文档和长方体对象
        self.doc = App.ActiveDocument if App.ActiveDocument else App.newDocument()
        self.box_obj = self.doc.addObject("Part::Feature", "DynamicBox")
        self.box_obj.Shape = Part.makeBox(10, 10, 10)

        # 初始化 UI
        self.initUI()

    def initUI(self):
        """初始化图形界面"""

        # 设置窗口属性
        self.setWindowTitle('东台中学-机械臂维度控制器')
        self.setGeometry(300, 300, 300, 100)

        # 创建水平滑动条
        self.slider = QtGui.QSlider(QtCore.Qt.Horizontal, self)
        self.slider.setMinimum(10)   # 最小值 10mm
        self.slider.setMaximum(150)  # 最大值 150mm
        self.slider.setValue(10)     # 初始值 10mm

        # 绑定事件：当滑动条数值改变时，调用 on_slide 函数
        self.slider.valueChanged.connect(self.on_slide)

        # 创建标签显示当前值
        self.label = QtGui.QLabel('X 轴长度：10mm', self)

        # 设置垂直布局
        layout = QtGui.QVBoxLayout()
        layout.addWidget(self.label)
        layout.addWidget(self.slider)
        self.setLayout(layout)

    def on_slide(self, val):
        """滑动条数值改变时的回调函数"""
        # 核心逻辑：UI 数值实时映射到几何体属性
        self.box_obj.Shape = Part.makeBox(val, 10, 10)
        self.label.setText(f'X 轴长度：{val}mm')

        # 重新计算并刷新视图
        self.doc.recompute()

# 启动我们的专属控制台！
my_gui = SliderWindow()
my_gui.show()
```

### 代码详解

| 代码 | 说明 |
|------|------|
| `class SliderWindow(QtGui.QWidget)` | 定义一个窗口类，继承自 QWidget |
| `super().__init__()` | 调用父类的初始化方法 |
| `QtGui.QSlider(QtCore.Qt.Horizontal)` | 创建水平滑动条 |
| `setRange(min, max)` | 设置滑动条范围 |
| `valueChanged.connect(func)` | 绑定数值改变事件到回调函数 |
| `makeBox(val, 10, 10)` | 用滑动条的值创建长方体 |

### 架构认知

这段代码展示了：
1. **面向对象编程 (OOP)**：使用 `class` 封装功能

### 异常排查

如果滑动条拖动过快导致软件卡顿：
- 原因：主线程被密集的 `recompute()` 占用
- 解决：引入**节流 (Throttling)** 机制，限制刷新频率

## 🚀 拓展挑战

### 挑战 1：添加第二个滑动条

```python
class MultiSliderWindow(QtGui.QWidget):
    def __init__(self):
        super().__init__()
        self.doc = App.ActiveDocument if App.ActiveDocument else App.newDocument()
        self.box_obj = self.doc.addObject("Part::Feature", "DynBox")
        self.box_obj.Shape = Part.makeBox(10, 10, 10)
        self.initUI()

    def initUI(self):
        self.setWindowTitle('三维控制器')
        self.setGeometry(300, 300, 350, 150)

        # X 轴滑动条
        self.slider_x = QtGui.QSlider(QtCore.Qt.Horizontal, self)
        self.slider_x.setRange(10, 100)
        self.slider_x.setValue(10)
        self.slider_x.valueChanged.connect(self.on_slide)

        # Y 轴滑动条
        self.slider_y = QtGui.QSlider(QtCore.Qt.Horizontal, self)
        self.slider_y.setRange(10, 100)
        self.slider_y.setValue(10)
        self.slider_y.valueChanged.connect(self.on_slide)

        # 布局
        layout = QtGui.QVBoxLayout()
        layout.addWidget(QtGui.QLabel('X 轴：'))
        layout.addWidget(self.slider_x)
        layout.addWidget(QtGui.QLabel('Y 轴：'))
        layout.addWidget(self.slider_y)
        self.setLayout(layout)

    def on_slide(self, val):
        x = self.slider_x.value()
        y = self.slider_y.value()
        self.box_obj.Shape = Part.makeBox(x, y, 10)
        self.doc.recompute()

my_gui = MultiSliderWindow()
my_gui.show()
```

### 挑战 2：控制旋转角度

```python
def on_slide(self, val):
    # 将滑动条值映射到旋转角度（0-360度）
    angle_deg = val  # 滑动条 0-360
    angle_rad = math.radians(angle_deg)  # 转换为弧度

    # 设置旋转（绕 Z 轴）
    self.box_obj.Placement.Rotation.setAxisAngle(
        App.Vector(0, 0, 1),  # Z 轴
        angle_rad
    )
    self.doc.recompute()
```

### 挑战 3：旋转控制面板

```python
class RotationWindow(QtGui.QWidget):
    def __init__(self):
        super().__init__()
        self.doc = App.ActiveDocument if App.ActiveDocument else App.newDocument()
        self.box_obj = self.doc.addObject("Part::Feature", "RotBox")
        self.box_obj.Shape = Part.makeBox(20, 20, 20)
        self.initUI()

    def initUI(self):
        self.setWindowTitle('旋转控制器')
        self.setGeometry(300, 300, 250, 120)

        # X 轴旋转
        self.slider_x = QtGui.QSlider(QtCore.Qt.Horizontal, self)
        self.slider_x.setRange(0, 360)
        self.slider_x.valueChanged.connect(self.on_rotate_x)

        layout = QtGui.QVBoxLayout()
        layout.addWidget(QtGui.QLabel('X 轴旋转：'))
        layout.addWidget(self.slider_x)
        self.setLayout(layout)

    def on_rotate_x(self, val):
        angle = math.radians(val)
        self.box_obj.Placement.Rotation.setAxisAngle(
            App.Vector(1, 0, 0), angle
        )
        self.doc.recompute()

rot_gui = RotationWindow()
rot_gui.show()
```

## 💡 知识点总结

| 概念 | 说明 |
|------|------|
| `QtGui.QWidget` | Qt 的基础窗口类 |
| `QtGui.QSlider` | 滑动条控件 |

| `valueChanged.connect(func)` | 绑定数值改变事件 |
| `setAxisAngle(vector, angle)` | 设置旋转轴和角度 |
| `math.radians(deg)` | 将角度转换为弧度 |
| 线性映射 | UI 数值到物理属性的数学转换 |
