## 📍 课时 9：自然韵律 —— 三角函数与波浪面模拟

### 物理/数理原理

在自然界中，声波、水波以及电磁波的传播都遵循着周期性规律。在数学上，我们使用正弦（$y = A \sin(\omega x + \phi)$）和余弦函数来描述这种简谐振动。

* **$A$ (振幅)**：决定波浪的起伏高度。
* **$\omega$ (角频率)**：决定波浪的密集程度（波长）。

利用空间坐标系的映射，我们可以将一维的三角函数转化为三维空间中的连绵起伏的波浪实体，这在声学降噪板或建筑美学设计中非常常见。

### FreeCAD 脚本

```python
import FreeCAD as App
import Part
import math
from FreeCAD import Vector

doc = App.ActiveDocument if App.ActiveDocument else App.newDocument("Wave")

points = []
amplitude = 5.0 # 振幅：5mm
frequency = 0.5 # 频率

# 利用for循环和三角函数计算波浪曲线上的点
for i in range(100):
    x = i
    z = amplitude * math.sin(frequency * x)
    points.append(Vector(x, 0, z))

# 将点连接成B样条曲线，并拉伸为曲面实体
wave_curve = Part.BSplineCurve()
wave_curve.buildFromPoles(points)
wave_edge = Part.Edge(wave_curve)
wave_face = Part.Face(Part.Wire([wave_edge])).extrude(Vector(0, 50, 0))

obj = doc.addObject("Part::Feature", "WaveSurface")
obj.Shape = wave_face
doc.recompute()
```

### 实验报告

* **观察记录**：执行脚本后，屏幕上生成了一块宽50mm、长100mm的波浪形曲面板。
* **变量探究**：将 `amplitude` 修改为 15.0，波峰明显变陡；将 `frequency` 修改为 0.1，波浪变得更加平缓宽阔。
* **STEAM思考**：如果在实际打印中采用这种结构作为墙面装饰，声波在不同频率的波浪面上反射时，会产生怎样的声学漫反射效果？

### 制作指南

1. 在 FreeCAD 中打开 `Python 控制台`。
2. 确保代码首部导入了 `math` 模块，这是 Python 进行高级数理计算的核心库。
3. 如果波浪线不够平滑，尝试将 `range(100)` 修改为更大数值，并相应减小步长，体验"微积分"中极限细分的思想。

## 📍 课时 12：维度控制器 —— QSlider滑动条与实时形变

### 物理/数理原理

在机械工程中，连杆和铰链的运动是连续的。为了在计算机中模拟这种连续的物理状态，我们需要建立UI控件数值与物体物理属性（如长度、旋转角度）之间的**线性映射关系**。

GUI滑动条提供的通常是离散的整数（如 0 到 100），我们需要通过公式将其转换为机械臂旋转所需的弧度（Radians）或确切的角度度数，实现"数据驱动物理形态"。

### FreeCAD 脚本

```python
from PySide import QtCore, QtGui
import FreeCAD as App
import Part

class SliderWindow(QtGui.QWidget):
    def __init__(self):
        super(SliderWindow, self).__init__()
        self.doc = App.ActiveDocument if App.ActiveDocument else App.newDocument()
        self.box_obj = self.doc.addObject("Part::Feature", "DynamicBox")
        self.box_obj.Shape = Part.makeBox(10, 10, 10)
        self.initUI()

    def initUI(self):
        self.setWindowTitle('东台中学-机械臂维度控制器')
        self.setGeometry(300, 300, 300, 100)

        # 创建水平滑动条
        self.slider = QtGui.QSlider(QtCore.Qt.Horizontal, self)
        self.slider.setMinimum(10) # 最小值 10mm
        self.slider.setMaximum(150) # 最大值 150mm

        # 绑定事件：当滑动条数值改变时，调用 on_slide 函数
        self.slider.valueChanged.connect(self.on_slide)

        layout = QtGui.QVBoxLayout()
        layout.addWidget(QtGui.QLabel("拖动以控制零件X轴伸缩："))
        layout.addWidget(self.slider)
        self.setLayout(layout)

    def on_slide(self, val):
        # 核心逻辑：UI数值实时映射到几何体属性
        self.box_obj.Shape = Part.makeBox(val, 10, 10)
        self.doc.recompute()

# 启动我们的专属控制台！
my_gui = SliderWindow()
my_gui.show()
```

### 实验报告

* **交互体验**：成功弹出独立窗口。用鼠标拖动滑动条时，3D视图中的长方体在X轴方向上如同具备生命般实时伸缩。
* **架构认知**：体验了面向对象编程（OOP）中 `Class` 的魅力，以及事件驱动机制（Event-Driven）在用户界面开发中的作用。
* **异常排查**：如果滑动条拖动过快导致软件卡顿，说明主线程被密集重绘占用，需要在大型复杂模型的刷新中引入节流（Throttling）机制。

### 制作指南

1. 本段代码涉及到 `PySide` 库的调用，这是 FreeCAD 原生集成的 Qt GUI 框架。
2. 注意类（Class）定义内部的缩进格式。
3. 高阶挑战：你能否添加第二个滑动条，将 `val` 绑定到 `self.box_obj.Placement.Rotation` 上，从而利用滑动条控制它的三维旋转？

## 📍 课时 16：空间碰撞 —— 边界框与物理干涉检测

### 物理/数理原理

在工业设计或机器人运动规划中，部件之间的"干涉（碰撞）"会导致严重的工程事故。

在算法领域，最基础且高效的碰撞检测是 **AABB算法**（Axis-Aligned Bounding Box，轴对齐边界框）。系统为每个复杂几何体计算出一个包含它的最小正方体盒子，通过判断这两个盒子在X、Y、Z三个轴上的投影区间是否发生重叠，即可快速得出两个物理构件是否发生了空间碰撞。

### FreeCAD 脚本

```python
import FreeCAD as App
import Part
from FreeCAD import Vector

doc = App.ActiveDocument

# 获取场景中预先创建好的两个测试零件（学生需提前在场景中放置两个物体）
actor = doc.getObject("Box") # 运动的物体
obstacle = doc.getObject("Cylinder") # 障碍物

# 获取两个物体的包围盒 (Bounding Box)
actor_bbox = actor.Shape.BoundBox
obstacle_bbox = obstacle.Shape.BoundBox

# 物理干涉判断
if actor_bbox.intersect(obstacle_bbox):
    App.Console.PrintMessage("【警告】发生空间碰撞！\n")
    # 发生碰撞，将障碍物变为警示红色
    obstacle.ViewObject.ShapeColor = (1.0, 0.0, 0.0)
else:
    App.Console.PrintMessage("安全距离。\n")
    # 安全状态，保持为绿色
    obstacle.ViewObject.ShapeColor = (0.0, 1.0, 0.0)

doc.recompute()
```

### 实验报告

* **现象验证**：手动改变 `Box` 的位置使其与 `Cylinder` 产生重叠，运行代码，控制台准确发出警告，且圆柱体瞬间变红。移开后再次运行，颜色恢复绿色。
* **算法局限性**：由于使用的是包围盒（BoundBox），即使两个不规则物体（如两个相近的星形）实际上并未触碰，但只要它们的"隐形方盒"重叠了，系统也会判定为碰撞。这种妥协是为了换取极高的运算效率。
* **实际应用**：在课题组设计的"机械臂抓取模拟"项目中，可以结合前一节的 QTimer 定时器，当机械臂触碰到零件时自动停止运动。

### 制作指南

1. 在运行此脚本前，务必在 FreeCAD 视图中创建一个名为 `Box` 的立方体和一个名为 `Cylinder` 的圆柱体。
2. 颜色的赋值使用的是 RGB 格式，`(1.0, 0.0, 0.0)` 代表纯红，范围是 0.0 到 1.0 之间。
3. 结合模块四的知识，试着为你写的碰撞检测逻辑套上一个"开始检测"的 GUI 按钮。
