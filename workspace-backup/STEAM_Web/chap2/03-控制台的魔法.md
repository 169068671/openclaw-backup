## 📐 物理/数理原理

### 代数中的"未知数"思想

在初中代数中，我们学过：
- 设 x 为未知数
- x = 5 表示 x 的值是 5
- 当 x 变化时，表达式的结果也会变化

在编程中，**变量**就是代数中的"未知数"：
- 变量是存储在计算机内存中的**数据容器**
- 改变容器里的值，实体的物理尺寸就会随之改变
- 这就是**参数化设计**的核心

### 变量命名规则

好的变量名应该：
- ✅ **有意义**：`height` 比 `h` 更好
- ✅ **英文单词**：避免使用拼音
- ✅ **小写字母**：Python 推荐小写
- ✅ **下划线分隔**：`box_height` 比 `boxHeight` 更符合 Python 风格

## 💻 FreeCAD 脚本

让我们用变量来控制圆柱体的尺寸：

```python
import Part

# 定义我们的魔法变量
my_radius = 15      # 圆柱的半径
my_height = 30      # 圆柱的高度

# 利用变量生成圆柱体
my_cylinder = Part.makeCylinder(my_radius, my_height)

# 显示圆柱体
Part.show(my_cylinder)

# 刷新视图
App.ActiveDocument.recompute()

# 打印信息
App.Console.PrintMessage("✅ 圆柱体创建成功！\n")
App.Console.PrintMessage(f"📐 半径：{my_radius}mm，高度：{my_height}mm\n")
```

### 代码详解

| 代码 | 说明 |
|------|------|
| `my_radius = 15` | 定义变量 `my_radius`，赋值为 15 |
| `my_height = 30` | 定义变量 `my_height`，赋值为 30 |
| `Part.makeCylinder(my_radius, my_height)` | 使用变量作为参数创建圆柱体 |
| `f"半径：{my_radius}mm"` | f-string 格式化字符串，嵌入变量值 |

## 🛠️ 制作指南

### 命名规范对比

```python
# ❌ 不好的命名（让人困惑）
a = 15
b = 30
c = Part.makeCylinder(a, b)

# ✅ 好的命名（清晰易懂）
radius = 15
height = 30
cylinder = Part.makeCylinder(radius, height)
```

### 变量的优势

**传统方式（硬编码）**：
```python
# 如果要改变圆柱尺寸，需要找到并修改所有数字
box1 = Part.makeBox(10, 10, 10)
box2 = Part.makeBox(10, 10, 10)
box3 = Part.makeBox(10, 10, 10)
# 如果想把尺寸从 10 改成 20，需要改 9 个数字！
```

**变量方式（参数化）**：
```python
# 只需修改一个变量，所有立方体都会改变
size = 20  # 只改这一行！
box1 = Part.makeBox(size, size, size)
box2 = Part.makeBox(size, size, size)
box3 = Part.makeBox(size, size, size)
```

## 🚀 拓展挑战

### 挑战 1：创建参数化笔筒

```python
import Part
from FreeCAD import Vector

# 参数化设计：只需修改这几个变量就能改变笔筒
holder_radius = 40    # 笔筒外半径
holder_height = 80    # 笔筒高度
wall_thickness = 3    # 壁厚
holder_sides = 6      # 边数（6=六边形）

# 创建外层
outer = Part.makePrism(
    Part.makePolygon([Vector(r * math.cos(i * 2 * math.pi / holder_sides),
                            r * math.sin(i * 2 * math.pi / holder_sides), 0)
                     for i in range(holder_sides)]),
    holder_height
)

# 创建内层挖空
inner_radius = holder_radius - wall_thickness
inner = Part.makeCylinder(inner_radius, holder_height)

# 布尔相减
result = outer.cut(inner)
Part.show(result)
```

### 挑战 2：变量类型探索

```python
# Python 有多种数据类型
name = "我的模型"      # 字符串 (str)
radius = 15.5          # 浮点数 (float)
count = 3              # 整数 (int)
is_visible = True      # 布尔值 (bool)
dimensions = [10, 20, 30]  # 列表 (list)

print("类型：", type(name))
print("类型：", type(radius))
print("类型：", type(count))
print("类型：", type(is_visible))
print("类型：", type(dimensions))
```

## 💡 知识点总结

| 概念 | 说明 |
|------|------|
| 变量 (Variable) | 存储数据的容器 |
| 赋值运算符 `=` | 将右边的值赋给左边的变量 |
| f-string | `f"半径：{radius}mm"`，在字符串中嵌入变量 |
| 参数化设计 | 用变量控制模型，修改变量即可改变模型 |
| 变量命名 | 使用有意义的英文单词，遵循小写+下划线规范 |
