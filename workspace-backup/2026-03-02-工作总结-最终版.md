## 🎯 关键决策

### 决策 1：使用 SSH 隧道代替 Tinyproxy

**原因：**
- Tinyproxy HTTPS 代理不稳定
- 经常超时和连接重置
- 用户体验差

**决策：**
- 切换到 SSH 隧道（SOCKS5）
- 更稳定、更可靠

### 决策 2：本地安装 Whisper + VPS 下载模型

**原因：**
- Docker 镜像拉取失败（网络问题）
- 模型直接下载失败（网络问题）

**决策：**
- 本地安装 faster-whisper（使用国内 pip 镜像）
- VPS 下载模型后传输到本地

### 决策 3：停用 SOCKS5，直接在 VPS 上访问国外网站

**原因：**
- SOCKS5 代理增加复杂度
- 需要持续维护
- VPS 访问更直接、更快

**决策：**
- 停用 SOCKS5 代理（22:44）
- 直接在 VPS 上访问国外网站
- 使用 scp 传输文件到本地
