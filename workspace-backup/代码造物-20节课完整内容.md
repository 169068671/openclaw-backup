# 📂 模块一：启航 —— STEAM与数字造物

## 📍 第 1 课：造物新纪元（3D打印与数字空间）

### 物理/数理原理

```python
import FreeCAD as App
```

## 📍 第 2 课：初探FreeCAD（代码召唤几何体）

### 物理/数理原理

任何复杂的三维物体，都可以被拆解为最基础的拓扑几何体（拓扑学概念）。正方体由长宽高决定，球体由半径决定。

### FreeCAD 脚本

```python
import Part
# 在坐标原点 (0,0,0) 召唤一个半径为 10 的基础球体
my_sphere = Part.makeSphere(10)
Part.show(my_sphere)
App.ActiveDocument.recompute() # 刷新视觉缓存
```

### 实验报告

控制台输入回车后，屏幕中央瞬间生成了一个灰色的标准球体。

### 制作指南

按住键盘 `Shift` 键 + 鼠标右键拖动，可以旋转我们的 3D 视角；滚动鼠标滚轮可以缩放。

## 📍 第 3 课：控制台的魔法（Python变量的引入）

### 物理/数理原理

代数中的"未知数"思想。在编程中，变量是存储在内存中的数据容器。改变容器里的值，实体的物理尺寸就会随之改变。

```python
import Part
# 定义我们的魔法变量
my_radius = 15
my_height = 30

my_cylinder = Part.makeCylinder(my_radius, my_height)

```

### 实验报告

成功生成了圆柱体。当我们把 `my_radius` 改为 5 时，圆柱体变成了一根细长的棍子。

### 制作指南

## 📍 第 4 课：参数化思维（空间位置的向量控制）

### 物理/数理原理

空间向量（Vector）。物体在三维空间中的平移，本质上是其各个顶点沿 X、Y、Z 轴的位移量叠加。

### FreeCAD 脚本

```python
from FreeCAD import Vector
import Part
box = Part.makeBox(10, 10, 10)
obj = App.ActiveDocument.addObject("Part::Feature", "MovingBox")
obj.Shape = box
# 操控向量，让方块在 X 轴前进 50 毫米
obj.Placement.Base = Vector(50, 0, 0)
App.ActiveDocument.recompute()
```

### 实验报告

方块没有在原点生成，而是精确地出现在了 X 轴 50mm 的位置。

### 制作指南

`Vector(X, Y, Z)` 是我们以后控制机械臂或零件装配最核心的数学工具。

## 📍 第 5 课：秩序的阵列（for 循环与线性构建）

### 物理/数理原理

等差数列。利用循环变量 `i`，每次乘以固定的步长，即可在空间中生成均匀排列的结构（如阶梯、栅栏）。

### FreeCAD 脚本

```python
from FreeCAD import Vector
# 用 for 循环建造 5 级台阶
for i in range(5):
    step = Part.makeBox(20, 10, 2)
    obj = App.ActiveDocument.addObject("Part::Feature", f"Step_{i}")
    obj.Shape = step
    # 高度(Z)和深度(Y)随 i 递增
    obj.Placement.Base = Vector(0, i*10, i*2)
```

### 实验报告

代码运行瞬间，5 级台阶如同多米诺骨牌一样精准排列在空间中，节省了手动拖拽的时间。

### 制作指南

注意 Python 的**缩进**（Tab 或 4个空格），缩进决定了哪些代码是在循环内部执行的。

## 📍 第 6 课：逻辑岔路口（if 条件判断与间隔挖空）

### 物理/数理原理

布尔逻辑与求余运算（模运算 `%`）。通过判断序号是奇数还是偶数，决定空间的物质是"填充"还是"留空"。

### FreeCAD 脚本

```python
for i in range(10):
    if i % 2 == 0: # 如果是偶数
        box = Part.makeBox(5, 5, 5)
        obj = App.ActiveDocument.addObject("Part::Feature", f"Block_{i}")
        obj.Shape = box
        obj.Placement.Base = Vector(i*10, 0, 0)
        # 奇数位置留空，形成虚实结合的城墙效果
```

### 实验报告

生成了类似长城城墙顶部的凹凸结构。

### 制作指南

`==` 是判断是否相等，而 `=` 是赋值。在写 if 语句时千万不要漏掉末尾的冒号 `:`。

## 📍 第 8 课：封装造物术（def 函数与模块化设计）

### 物理/数理原理

系统工程中的"模块化封装"思想。将复杂的机械结构封装成一个具有输入参数的黑盒（函数），提高复用率。

### FreeCAD 脚本

```python
# 定义一个制造带孔方块的"车床"函数
def make_hollow_box(size, hole_radius):
    outer = Part.makeBox(size, size, size)
    inner = Part.makeCylinder(hole_radius, size)
    # 布尔运算：减去中间的圆柱
    result = outer.cut(inner)
    Part.show(result)

# 呼叫我们的车床，立刻生产两个不同规格的零件
make_hollow_box(10, 3)
make_hollow_box(20, 8)
```

### 实验报告

利用自定义函数，我们只需一行代码就能完成"画方块-画圆柱-切除"的复杂布尔运算流程。

### 制作指南

在设计复杂产品（如汽车）时，每一个螺丝钉都可以写成一个函数。

## 📍 第 9 课：自然韵律（三角函数与波浪面模拟）

### 物理/数理原理

简谐振动。正弦（Sine）和余弦（Cosine）函数描述了自然界中的声波、水波以及电磁波的周期性变化。

### FreeCAD 脚本

```python
import math
points = []
# 利用三角函数计算波浪坐标
for i in range(50):
    x = i
    z = 5 * math.sin(i * 0.5) # 振幅5，角频率0.5
    points.append(Vector(x, 0, z))

# 拟合平滑的B样条曲线
wave = Part.BSplineCurve()
wave.buildFromPoles(points)
Part.show(wave.toShape())
```

### 实验报告

生成了一条优美的正弦波浪线，通过修改振幅和频率参数，可以模拟平静的水面或汹涌的海浪。

### 制作指南

必须导入 `math` 模块。如果将这条线沿 Y 轴拉伸（Extrude），就能得到一块声学降噪墙板。

## 📍 第 10 课：分形与迭代（算法生成艺术）

### 物理/数理原理

分形几何（Fractal）。局部与整体具有自相似性，通过算法的递归或多层嵌套迭代，用极简规则生成无限复杂的形态。

### FreeCAD 脚本

```python
# 基础迭代思想：嵌套循环生成网格阵列
for x in range(0, 30, 10):
    for y in range(0, 30, 10):
        # 留出中间的空洞，形成类似谢尔宾斯基地毯的雏形
        if x != 10 or y != 10:
            box = Part.makeBox(8, 8, 8)
            box.translate(Vector(x, y, 0))
            Part.show(box)
```

### 实验报告

生成了一个 3x3 的方块阵列，但中心位置被精确地留空了。

### 制作指南

# 📂 模块四：黄金时代 —— 掌控交互 (FreeCAD GUI 开发)

## 📍 第 11 课：告别黑框（PySide 基础与按钮）

### 物理/数理原理

事件驱动模型（Event-Driven）。GUI界面处于监听状态，当捕捉到用户的"点击"事件时，触发绑定的回调函数执行。

### FreeCAD 脚本

```python
from PySide import QtGui
class SimpleUI(QtGui.QWidget):
    def __init__(self):
        super().__init__()
        self.btn = QtGui.QPushButton('生成标准齿轮', self)
        self.btn.clicked.connect(self.make_gear) # 绑定事件
        layout = QtGui.QVBoxLayout()
        layout.addWidget(self.btn)
        self.setLayout(layout)
        self.show()

    def make_gear(self):
        Part.show(Part.makeCylinder(10, 2)) # 用圆柱示意齿轮坯

gui = SimpleUI()
```

### 实验报告

在 FreeCAD 中弹出了一个我们自己编写的按钮窗口。点击按钮，场景中自动生成了模型。

### 制作指南

`gui = SimpleUI()` 这一步必须实例化并保存在全局变量中，否则窗口会一闪而退被内存回收。

## 📍 第 12 课：维度控制器（QSlider 滑动条交互）

### 物理/数理原理

线性映射。将滑动条离散的整型数值（如 0-100）通过数学比例运算，映射为 3D 模型的物理尺寸或旋转弧度。

### FreeCAD 脚本

```python
from PySide import QtCore, QtGui
class SliderUI(QtGui.QWidget):
    def __init__(self):
        super().__init__()
        self.slider = QtGui.QSlider(QtCore.Qt.Horizontal, self)
        self.slider.setRange(10, 50) # 设置滑动范围 10mm 到 50mm
        self.slider.valueChanged.connect(self.update_model)
        layout = QtGui.QVBoxLayout()
        layout.addWidget(self.slider)
        self.setLayout(layout)
        self.obj = App.ActiveDocument.addObject("Part::Feature", "DynBox")
        self.show()

    def update_model(self, val):
        self.obj.Shape = Part.makeBox(val, 10, 10) # 实时修改长度
        App.ActiveDocument.recompute()

slider_gui = SliderUI()
```

### 实验报告

拖动滑动条，屏幕上的长方体如同拥有生命一般，长度随着鼠标的滑动实时伸缩。

### 制作指南

频繁的 `recompute()` 可能会导致画面卡顿，这是因为每次变化都在重新计算 3D 网格。

## 📍 第 13 课：数据录入舱（QLineEdit 文本与数值转换）

### 物理/数理原理

数据类型转换（Type Casting）。用户在界面输入的本质上是字符串（String），需要转换为浮点数（Float）才能参与几何尺寸计算。

### FreeCAD 脚本

```python
from PySide import QtGui
class InputUI(QtGui.QWidget):
    def __init__(self):
        super().__init__()
        self.input_box = QtGui.QLineEdit("15.5", self) # 默认值
        self.btn = QtGui.QPushButton('生成定制圆球', self)
        self.btn.clicked.connect(self.generate)
        layout = QtGui.QVBoxLayout()
        layout.addWidget(self.input_box)
        layout.addWidget(self.btn)
        self.setLayout(layout)
        self.show()

    def generate(self):
        # 将字符串转换为浮点数
        radius = float(self.input_box.text())
        Part.show(Part.makeSphere(radius))

input_gui = InputUI()
```

### 实验报告

我们成功实现了一个带有输入框的微型工业软件！输入精确的小数，就能生成对应尺寸的球体。

### 制作指南

如果在输入框里输入了字母，点击生成时 Python 会报错 `ValueError`。在真实的软件开发中，这里需要加 `try-except` 错误处理。

# 📂 模块五：铂金时代 —— 动力与模拟

## 📍 第 15 课：时间循环（QTimer 与自动动画）

### 物理/数理原理

角速度。齿轮或轮子在不受外力阻碍时，随时间推移按固定角度旋转。我们利用定时器剥离主线程，模拟时间的流逝。

### FreeCAD 脚本

```python
from PySide import QtCore
import math
from FreeCAD import Vector

# 假设场景中有一个名为 "Gear" 的物体
obj = App.ActiveDocument.getObject("Gear")
angle = 0

def rotate_gear():
    global angle
    angle += 5 # 每次增加5度
    obj.Placement.Rotation.setAxisAngle(Vector(0,0,1), math.radians(angle))
    App.ActiveDocument.recompute()

# 启动定时器，每 50 毫秒执行一次旋转
timer = QtCore.QTimer()
timer.timeout.connect(rotate_gear)
timer.start(50)
```

### 实验报告

运行后，无需手动操作，场景中的齿轮开始匀速自转，我们成功实现了动力学动画！

### 制作指南

使用 `timer.stop()` 可以停止动画。注意角度必须通过 `math.radians()` 转换为弧度制才能被物理引擎识别。

## 📍 第 16 课：空间碰撞（AABB 边界框干涉检测）

### 物理/数理原理

空间干涉。两个刚体不能占据同一个物理空间。算法中使用轴对齐边界框（AABB）来粗略但极速地判断两个物体是否发生接触。

### FreeCAD 脚本

```python
actor = App.ActiveDocument.getObject("MovingBox")
obstacle = App.ActiveDocument.getObject("Wall")

# 碰撞检测判定
if actor.Shape.BoundBox.intersect(obstacle.Shape.BoundBox):
    App.Console.PrintMessage("【警告】发生碰撞！\n")
    obstacle.ViewObject.ShapeColor = (1.0, 0.0, 0.0) # 变红
else:
    obstacle.ViewObject.ShapeColor = (0.0, 1.0, 0.0) # 绿色安全
```

### 实验报告

当移动的方块触碰到墙壁模型时，墙壁瞬间变成了红色。我们为虚拟世界加入了物理法则。

### 制作指南

BoundBox（包围盒）是一个包裹物体的隐形立方体，它是 3D 游戏开发中做碰撞检测的基础。

## 📍 第 17 课：轨迹追踪（B-Spline 与路径运动）

### 物理/数理原理

质点沿曲线的运动轨迹模拟。利用参数 $t$ (0 到 1)，获取曲线上特定比例处的切线向量和三维坐标，实现平滑移动。

### FreeCAD 脚本

```python
curve_obj = App.ActiveDocument.getObject("Path")

# 获取曲线上 50% 处的精确三维坐标
target_vector = curve_obj.Shape.Edge1.valueAt(0.5)

```

### 制作指南

## 📍 第 19 课：虚实跨越（切片策略与打印支撑）

### 物理/数理原理

重力与悬垂效应（FDM 3D打印机特性）。打印机喷嘴挤出的熔融塑料若下方悬空超过 45 度，将因重力塌陷，必须添加辅助支撑（Support）。

### FreeCAD 操作步骤

代码已跑通，本节进入实体阶段：

1. 在 FreeCAD 中选中最终模型
2. 点击 `文件 (File)` -> `导出 (Export)`
