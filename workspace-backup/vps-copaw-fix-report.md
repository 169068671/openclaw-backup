## 🛡️ 预防措施

### 更新 AGENTS.md

在 `/root/.copaw/AGENTS.md` 中添加了新的"防止重复写入"规则：

#### 绝对禁止的行为

- 不要循环读取同一文件并追加写入
- 不要在没有检查的情况下重复写入相同内容
- 不要逐字符或逐行重复文本
- 写入前必须检查文件大小

#### 安全的写入流程

1. **读取现有内容** - 使用 read_file 读取目标文件
2. **检查是否已存在** - 使用 grep 检查要写入的内容
3. **检查文件大小** - 如果文件 >100KB，停止写入
4. **只写入新内容** - 只添加不存在的新信息
5. **验证写入结果** - 写入后立即读取确认

#### 记忆文件大小限制

- MEMORY.md: 建议不超过 100KB
- memory/YYYY-MM-DD.md: 建议不超过 50KB

#### 出现重复写入的迹象

- 文件大小快速增长（>1MB）
- 同一内容重复多次出现
- grep -c 显示同一模式出现 >10 次
