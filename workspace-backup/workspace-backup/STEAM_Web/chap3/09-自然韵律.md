## 📐 物理/数理原理

### 简谐振动

在自然界中，许多现象都遵循**周期性规律**：
- 🌊 水波的起伏
- 🔔 声音的传播
- 💡 光波的振动
- 🎸 弦乐的震动

数学上，我们用**三角函数**来描述这种**简谐振动**：
- **正弦函数**：`y = A · sin(ωx + φ)`
- **余弦函数**：`y = A · cos(ωx + φ)`

### 参数含义

| 参数 | 名称 | 作用 |
|------|------|------|
| A | 振幅 (Amplitude) | 决定波浪的起伏高度 |
| ω (omega) | 角频率 (Angular Frequency) | 决定波浪的密集程度（波长） |
| φ (phi) | 初相位 (Phase Shift) | 决定波浪的起始位置 |

### 空间映射

将一维的三角函数**映射**到三维空间：
- X 轴：波浪的延伸方向
- Y 轴：波浪的宽度
- Z 轴：波浪的起伏（由三角函数计算）

这种结构在工程中很常见：
- 🏗️ 声学降噪板（漫反射声波）
- 🎨 建筑美学设计（流线型外观）
- 🌪️ 风洞实验模型（空气动力学）

## 💻 FreeCAD 脚本

让我们用三角函数创建一条优美的波浪线：

```python
import FreeCAD as App
import Part
import math
from FreeCAD import Vector

# 创建新文档
doc = App.ActiveDocument if App.ActiveDocument else App.newDocument("Wave")

# 定义波浪参数
amplitude = 5.0   # 振幅：5mm
frequency = 0.5   # 频率

# 存储波浪曲线上的点
points = []

# 利用 for 循环和三角函数计算波浪坐标
for i in range(100):
    x = i                      # X 坐标：线性递增
    z = amplitude * math.sin(frequency * x)  # Z 坐标：正弦波动
    points.append(Vector(x, 0, z))  # 添加到点列表

# 将点连接成 B 样条曲线
wave_curve = Part.BSplineCurve()
wave_curve.buildFromPoles(points)

# 转换为线段
wave_edge = Part.Edge(wave_curve)

# 沿 Y 轴拉伸成曲面
wave_face = Part.Face(Part.Wire([wave_edge])).extrude(Vector(0, 50, 0))

# 创建对象并显示
obj = doc.addObject("Part::Feature", "WaveSurface")
obj.Shape = wave_face

# 刷新视图
doc.recompute()

# 打印信息
App.Console.PrintMessage("✅ 波浪面生成成功！\n")
App.Console.PrintMessage(f"📐 振幅：{amplitude}mm，频率：{frequency}\n")
App.Console.PrintMessage(f"📏 尺寸：100mm × 50mm\n")
```

### 代码详解

| 代码 | 说明 |
|------|------|
| `import math` | 导入数学模块，提供三角函数 |
| `math.sin(x)` | 计算正弦值 |
| `points = []` | 创建空列表，存储坐标点 |
| `points.append(Vector(x, y, z))` | 添加一个三维点到列表 |
| `Part.BSplineCurve()` | 创建 B 样条曲线（平滑曲线） |
| `buildFromPoles(points)` | 从点列表构建曲线 |
| `extrude(Vector(dx, dy, dz))` | 沿指定向量拉伸 |

## 🛠️ 制作指南

### 关键步骤

1. **导入 math 模块**：必须导入才能使用三角函数
2. **设置采样点数量**：`range(100)` 表示 100 个点，越多越平滑
3. **计算坐标**：X 轴线性增长，Z 轴用 `sin` 函数计算
4. **构建曲线**：使用 B 样条曲线，比折线更平滑
5. **拉伸成面**：将曲线沿 Y 轴拉伸成曲面

### 调试技巧

如果波浪不够平滑：
```python
# 方法1：增加采样点
for i in range(200):

# 方法2：减小步长
for i in range(0, 100, 0.5):  # 错误！range 需要整数
# 正确做法：
import numpy as np
for i in np.arange(0, 100, 0.5):  # 使用 numpy
```

## 🚀 拓展挑战

### 挑战 1：创建双层波浪

```python
# 两个波浪叠加，创造更复杂的形态
for i in range(100):
    x = i
    # 波浪1 + 波浪2
    z = 5 * math.sin(0.5 * x) + 3 * math.cos(0.8 * x)
    points.append(Vector(x, 0, z))
```

### 挑战 2：创建三维波浪面

```python
# 网格状的三维波浪
for x in range(0, 100, 5):
    for y in range(0, 50, 5):
        z = 3 * math.sin(0.3 * x) * math.cos(0.3 * y)
        # 在每个位置创建一个小方块模拟波浪
        box = Part.makeBox(4, 4, 4)
        box.translate(Vector(x, y, z))
        Part.show(box)
```

### 挑战 3：用余弦函数

```python
# 余弦函数（相位偏移 π/2）
for i in range(100):
    x = i
    z = amplitude * math.cos(frequency * x)
    points.append(Vector(x, 0, z))
```

## 💡 知识点总结

| 概念 | 说明 |
|------|------|
| `math.sin(x)` | 正弦函数，值域 [-1, 1] |
| `math.cos(x)` | 余弦函数，值域 [-1, 1] |
| 振幅 A | 决定波浪的起伏高度 |
| 频率 ω | 决定波浪的密集程度 |
| B 样条曲线 | 平滑曲线，由控制点拟合 |
| `extrude(vector)` | 将二维图形拉伸成三维实体 |
