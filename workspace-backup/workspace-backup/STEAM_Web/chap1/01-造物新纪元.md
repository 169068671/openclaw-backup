## 📐 物理/数理原理

### 增材制造（FDM）物理过程

3D 打印的本质是**增材制造**（Additive Manufacturing），与传统的"减材制造"（如车、铣、钻）相反。

在 FDM（熔融沉积建模）技术中：
1. **材料受热熔化**：塑料丝材（如 PLA、ABS）在喷嘴中被加热至熔融状态（约 200°C）
2. **重力与粘性作用**：熔融材料在重力和粘性作用下挤出
3. **三维笛卡尔坐标系**：喷嘴按照 X/Y/Z 三轴坐标逐层堆叠
4. **冷却固化**：材料与先前层接触后快速冷却，形成实体

### 三维笛卡尔坐标系

FreeCAD 中的三维空间遵循笛卡尔坐标系：
- **X 轴**：左右方向
- **Y 轴**：前后方向
- **Z 轴**：上下方向（通常代表高度）

每个点都可以用 `(x, y, z)` 坐标表示，原点 `(0, 0, 0)` 称为"世界原点"。

## 💻 FreeCAD 脚本

让我们用 Python 创建数字世界的第一块"画布"：

```python
import FreeCAD as App

# 建立我们在数字世界的第一块"画布"
doc = App.newDocument("STEAM_Project_01")

# 在控制台打印成功信息
App.Console.PrintMessage("✅ 创客空间初始化成功！\n")
App.Console.PrintMessage("📍 当前文档名称：" + doc.Name + "\n")
```

### 代码详解

- `import FreeCAD as App`：导入 FreeCAD 模块，简写为 `App`
- `App.newDocument("名称")`：创建一个新文档
- `App.Console.PrintMessage()`：在 FreeCAD 报告视图中输出信息

## 🛠️ 制作指南

### 第一步：调出控制台

1. 启动 FreeCAD
2. 点击顶部菜单栏 `视图 (View)` → `面板 (Panels)`
3. 勾选 ✅ `Python 控制台 (Python console)`
4. 勾选 ✅ `报告视图 (Report view)`

### 第二步：输入代码

在 Python 控制台中（`>>>` 提示符后）逐行输入：

```python
import FreeCAD as App
doc = App.newDocument("STEAM_Project_01")
App.Console.PrintMessage("创客空间初始化成功！\n")
```

**⚠️ 注意事项**：
- 代码中的所有标点符号必须是**英文半角**（`,` `.` `(` `)` `"`）
- 字符串要加引号（单引号或双引号都可以）
- 不要漏掉最后的 `\n`（表示换行）

### 第三步：查看结果

观察 FreeCAD 界面：
- 新建文档成功后，界面上方会出现文档选项卡
- 3D 视图是空的（这是正常的，因为我们还没创建几何体）
- 报告视图（底部）会显示我们输出的信息

## 🚀 拓展挑战

### 挑战 1：创建多个文档

尝试创建多个文档，看看会发生什么：

```python
doc1 = App.newDocument("Project_A")
doc2 = App.newDocument("Project_B")
doc3 = App.newDocument("Project_C")
```

### 挑战 2：获取文档信息

探索 FreeCAD 的 API，获取更多信息：

```python
doc = App.newDocument("TestDoc")
print("文档数量：", len(App.listDocuments()))
print("当前文档名称：", doc.Name)
print("当前文档是否活跃：", App.ActiveDocument == doc)
```

### 挑战 3：保存文档

学习如何保存文档到硬盘：

```python
doc = App.newDocument("SaveTest")
# 方法1：另存为
doc.saveAs("/tmp/MyFirstProject.FCStd")
# 方法2：保存（如果已经保存过）
doc.save()
App.Console.PrintMessage("文档已保存！\n")
```
