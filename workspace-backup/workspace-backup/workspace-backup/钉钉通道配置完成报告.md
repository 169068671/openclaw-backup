## 📱 使用方式

### 用户侧

**添加机器人：**
1. 在钉钉中搜索机器人（通过 AppKey 或应用名称）
2. 发起单聊或添加到群聊

**会话命令：**
- `/new`、`/reset`、`/clear` - 开启新会话
- `新会话`、`重新开始`、`清空对话` - 开启新会话

**与 OpenClaw 对话：**
直接发送消息，OpenClaw 会实时回复（打字机效果）

### OpenClaw 侧（主动推送）

**发送到用户：**
```bash
message --channel dingtalk-connector --to "用户ID" --message "消息内容"
```

**发送到群组：**
```bash
message --channel dingtalk-connector --to "群ID" --message "消息内容"
```
