```bash

# 转录音频
python3 ~/whisper-env/transcribe.py audio.mp3 small zh

# 运行测试
bash ~/whisper-env/test.sh
```

### 代理使用

```bash

# 检查隧道状态

```

### 钉钉使用

**在钉钉中直接对话即可，无需额外操作。**

### Docker 使用

```bash
# 拉取镜像（使用 VPS 下载规则）
# 1. 在 VPS 上下载
ssh root@76.13.219.143
docker pull ubuntu:latest
docker save ubuntu:latest -o ubuntu-latest.tar

# 2. 传输到本地
scp root@76.13.219.143:/root/ubuntu-latest.tar ~/

# 3. 本地导入
docker load -i ~/ubuntu-latest.tar
```
